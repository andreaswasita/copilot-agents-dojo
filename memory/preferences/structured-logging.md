# Structured Logging with Context

## What the User Prefers

Always use structured logging (JSON format) with contextual fields (`userId`, `requestId`, `action`) rather than string interpolation. Logs must be machine-parseable.

## How It Was Learned

Corrected twice when using `console.log(\`User ${id} failed\`)` instead of structured logger.

## When to Apply

- All backend services, API handlers, background jobs
- Not required for CLI scripts or one-off utilities
