# Memory Vault — Map of Content

This is the persistent knowledge graph for all Copilot agents in this repo.
Agents MUST read this file at session start to load relevant context.

## How This Works

- **Files** use YAML frontmatter for metadata + standard markdown links for connections
- **Links** use relative paths, e.g. `[title](decisions/chose-postgres.md)`
- **Backlinks** are auto-generated by `scripts/link-index.sh` at the bottom of each file
- **Graph** is stored in `memory/.link-graph.json` for programmatic queries

## Categories

### [Decisions](decisions/)
Architectural and design decisions with context, alternatives considered, and rationale.
Template: `decisions/_template.md`

### [Patterns](patterns/)
Recurring lessons promoted from `tasks/lessons.md` when they hit 3+ occurrences.
These are proven, validated rules — not one-off notes.
Template: `patterns/_template.md`

### [Preferences](preferences/)
Agent behavioral preferences learned from user corrections over time.
Not hardcoded rules — these emerge from working together.
Template: `preferences/_template.md`

### [Sessions](sessions/)
Session summaries linking back to decisions and patterns created or referenced.
Provides temporal context: "what happened when?"
Template: `sessions/_template.md`
